## 📋 What is NekoChat?

NekoChat is a chat plugin made for Minecraft server software like Paper, Spigot, Purpur, and Folia. It lets players talk in local areas, global chat, or private messages. The system also supports mentions, making communication clear and organized.

You do not need any programming skills to use it. Simply follow these steps to get it working.

Before installing NekoChat, make sure your system meets these requirements:

## 🚀 Installing NekoChat on Your Minecraft Server

1. **Find your server folder**  
   Locate the folder where your Minecraft server files live on your Windows computer.

2. **Open the plugins folder**  
   Inside your server folder, open (or create) the `plugins` folder. This is where all Minecraft plugins go.

3. **Move the NekoChat `.jar` file**  
   Place the NekoChat `.jar` file you downloaded into the `plugins` folder.

4. **Start your server**  
   Run your Minecraft server as usual. The server will load NekoChat automatically.

5. **Check if NekoChat loaded**  
   Look at your server console. If NekoChat is working, you will see a message confirming the plugin has started.

## ⚙️ Setting Up NekoChat for First Use

NekoChat works well straight out of the box, but you may want to make small changes.

- **Configure `config.yml`**  
  Inside the `plugins/NekoChat` folder, open the `config.yml` file with a text editor like Notepad.

  Here you can change settings like chat radius, message formats, and channels.

- **Use commands in Minecraft**  
  You can use simple chat commands to manage NekoChat while in the game. For example, `/nekochat help` will list common commands.

No technical knowledge is needed to edit basic settings. Just read the comments inside the `config.yml` file. They explain what each option does.

## 🎮 How to Use NekoChat in Minecraft

NekoChat adds features for players to chat easily:

- **Local chat**  
  Speak within a limited radius. Players close to you will hear you.

- **Global chat**  
  Send messages to all players online.

- **Mentions**  
  Tag someone by typing `@username`. The user will get a notification.

- **Private messages**  
  Use commands like `/msg username Hello` to send private messages.

The commands are easy to learn and will help keep conversations clear and smooth.
